Exercise 1: Single page app in Angular (4 CLI)

Tic-Tac-Toe Game:
------------------------------------------------------------------------------
Running instructions:

open cmd

>> print cd  '/..../tic-tac-toe'          // path to game project

>> npm install

>> ng serve

open your browser in http://localhost:4200/

and start playing :)

--------------------------------------------------------------------------------
for start playing :
1.click on 'Play Game' in navigaton bar
2.choose game mode from opening menu list :

The game is divided to 3 modes :
1. Basic game for 2 Human players.
2. Computer game for Human player against computer player.
3. Random Players game for 2 Humans who are requested from 'random user api'.

Enjoy your game !
